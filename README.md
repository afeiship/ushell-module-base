# ushell-module-base
> Unix like shell base module.

## installation
```shell
npm i @jswork/ushell-module-base
```

## dependencies
```shell
npm i -g del-cli
npm i -g ntl
npm i -g @jswork/git-url-cli
npm i -g npm-install-peers publish-preview yarn
```

## clipy
```shell
git clone git@github.com:afeiship/clipy.git
npm i
npm link
```
